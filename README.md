# my_gemini


A modern AI-powered chat assistant built using **HTML, Tailwind CSS, JavaScript, and Google Gemini API**.
This project provides a sleek multi-turn conversational interface with chat history, model selection, Markdown rendering, and responsive UI.

---

## 🚀 Features

* Multi-turn AI conversation support
* Google Gemini API integration
* Modern responsive UI using Tailwind CSS
* Markdown response rendering
* Real-time typing indicator
* Chat history management
* Model selection support
* Clean dark-themed interface
* Client-side only application

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* Tailwind CSS
* JavaScript

### APIs & Libraries

* Google Gemini API
* Marked.js
* Font Awesome

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/shivani-009/my_gemini.git
```

### 2. Open Project Folder

```bash
cd my_gemini
```

### 3. Run Project

Simply open:

```bash
index.html
```

OR use VS Code Live Server for better experience.

---

## 🔑 Configure Gemini API Key

Inside the application, enter your Gemini API key in the API Key input field.

Get your API key from:

https://aistudio.google.com/app/apikey
